# 🎮 ArcadeEater – 2D Arcade Game (C / SDL)

ArcadeEater is a desktop-based 2D arcade game developed using the C programming language.  
The project demonstrates structured game development concepts including modular programming, sprite rendering, and level management using the SDL (Simple DirectMedia Layer) library.

This project highlights low-level programming skills, graphics handling, and organized code architecture in C.

---

## 📌 Project Overview

ArcadeEater is a classic arcade-style game where the player navigates through levels using controlled movement and sprite-based rendering.

The project demonstrates:

- Modular programming in C
- 2D graphics rendering using SDL
- Map and level handling logic
- Resource and asset management
- Structured header and source file organization

---

### Core Language

## 📂 Project Structure

```
ArcadeEater/
│
├── main.c                # Entry point of the game
├── map.c                 # Map and level handling logic
├── move_guide.c          # Player movement logic
├── accessories.c         # Game utilities and accessories
├── header.h              # Function declarations and macros
├── resource.rc           # Resource configuration
│
├── sprites/              # Game sprite assets
├── obj/                  # Object files
├── bin/                  # Compiled binaries
│
├── SDL.dll               # SDL runtime library
├── SDL_image.dll         # SDL image handling library
├── jpeg.dll              # JPEG support library
│
└── ArcadeEater.exe       # Executable game file
```
